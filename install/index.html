ESP32-S3 Matrix — Install

The board and your computer must be on the same local network. After flashing, the board is reachable at http://esp32matrix.local.

No button? Your browser lacks Web Serial — use the offline method below.

1b. Flash the firmware (offline / other browsers)

Download the release, plug in the board, and run flash.bat (Windows) or flash.sh (macOS/Linux).

2. Add it to Claude Desktop

Download esp32-matrix.mcpb and double-click it — Claude Desktop installs the extension. No configuration needed.
