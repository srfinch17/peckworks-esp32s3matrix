The bigger idea

One message. Any screen.

Under the panel is a tiny protocol. Claude sends one semantic message — an intent, maybe some data — and any display you've connected renders it its own way. The 8×8 is just the first renderer.

// Claude says it once —
presence.set({ intent: "working", headline: "Running tests", data: { progress: 0.6 } })

Get it

Real hardware. Fully open.

It runs on a low-cost Waveshare ESP32-S3-Matrix (a ~$15 board, 8×8 WS2812B). The board and your computer just need to be on the same Wi-Fi. Three steps:

01 · Flash

One file to the board

Download the release and run flash.bat / flash.sh. One merged image — firmware and the web UI together, no extra upload. A one-click browser flasher is coming once this site is hosted.

02 · Connect

Add it to Claude Desktop

Double-click the .mcpb extension. No JSON, no Node install, no config — the board lives at esp32matrix.local on your network.

03 · Say hi

Watch it come alive

Claude starts using the panel on its own — a spinner when it's busy, a check when it's done, a tap when it needs you.

Where it stands: the firmware, the web control panel, and the Claude integration all work and are hardware-verified. The offline flasher is ready now; the one-click browser install goes live when this page is published. Full instructions and downloads are on GitHub.
