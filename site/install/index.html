Flash it in one click.

Plug the board into your computer over USB-C and flash the latest firmware straight from this page. Takes about a minute; afterwards the panel is yours at esp32matrix.local.

Install the firmware

Your browser doesn't support Web Serial. Use the offline installer below. You must use HTTPS to flash. Open this page at its github.io address.

Uses ESP Web Tools over Web Serial. Nothing is uploaded anywhere; the firmware is written from your browser directly to the board.

Alternative · any OS, any browser

Offline installer

Prefer not to use Chrome, or flashing many boards? Grab the offline bundle from the latest release: a single merged binary plus flash.bat (Windows) and flash.sh (macOS/Linux).

Download the latest release ↗

Unzip, plug in the board, and run the script for your OS. Windows includes esptool.exe; macOS/Linux need pip install esptool.

2 · After flashing

Get it on your WiFi

  • The board reboots and opens a setup hotspot named ESP32-Matrix-Setup.
  • Join it from your phone/laptop, pick your network, and enter the password.
  • It reconnects and is reachable at http://esp32matrix.local, then open that for the control panel.
  • Hold BOOT at power-on any time to wipe WiFi and start over.
Heads up: the distributed firmware ships with no WiFi credentials; you set them on first boot. Your network details stay on your board.
